Reup Privacy Policy

Effective date: April 16, 2026

Reup is a Chrome extension that detects visible reset messages on supported AI websites and notifies users when access may be available again.

What Reup accesses

Reup may access supported page URLs/domains and visible page text necessary to detect reset-related messages.

How data is used

Reup uses this information only to detect reset times, save user preferences locally, and send notifications.

Data storage

Reup stores settings and detected reset times locally in the user’s browser.

Data sharing

Reup does not sell user data and does not transfer user data to third parties for advertising or unrelated purposes.
