Reup Chrome Extension

How to load it:
1. Open Chrome.
2. Go to chrome://extensions.
3. Turn on Developer mode.
4. Click "Load unpacked".
5. Select the reup-extension-v6 folder.
6. Pin Reup so the popup stays easy to reach.

What it does:
- Watches ChatGPT, Claude, and Codex web pages in Chrome.
- Looks for visible reset-time text.
- Saves detected reset times locally.
- Sends an early reminder and a reset alert when the saved time arrives.
- Lets you keep the banner alert, the Reup chime, both, or neither.

Notes:
- Detection is heuristic and may need updates if the sites change their wording.
- Reup stores settings and detected reset times locally in Chrome.
- This project is for Chrome web pages only, not desktop apps or CLI tools.
